Software Required: Latest version of Jupyter Notebook

Please follow the below instructions to run the Notebook (Capital_One.ipynb) file.

Step 1: The main file to run is called Capital_One.ipynb. 
		 
	First upload the datasets airbnb in the (data->raw) folder of the submission folder. Capital_One.ipynb file is already present in notebook folder do not change the
        path of anyfile. Just open the folder in Jupyter Notebook and start running each cell of Capital_One.ipynb. 

Step 2: If required package is not installed then first installed the required package which is used in this project.


Step 3: You can run the notebook document step-by-step (one cell a time) or you can run the whole notebook in a single step by clicking on the menu cell-> Run All.
